My Portfolio

Rogue Pickings Website

HTML & CSS from Photoshop Design Comp

The Rogue Pickings website was a mock portfolio sample client site as part of our final project. Based on a Photoshop PSD comp, it consists of four sections, including navigation, a responsive full-width image, and a three-column portion for the specials, a review, and contact information. It also features an About page.

Jubilee Austen Project

HTML5 CSS & Website

The Jubilee Austen single-page website mock portfolio project took us from coding a site based on a Photoshop PSD to launching it. It introduced us to using a text editor, Photoshop, setting up structure and content with HTML, styling with CSS, validating and testing our code, obtaining a domain, and ultimately launching the website.

Holstee Manifesto

HTML5 & CSS Exercise

The Holstee Manifesto project was a fun, creative exercise allowing us to play with color, fonts, spans, and show off our floating skills. The pastel colors were inspired by nature in Spring.

About Me

Let me create this adventure for you!
